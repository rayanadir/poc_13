# P13-PoC


## Database setup
Go to folder:

> `cd resources`

Run script in `script.sql`


## Start the project

### Front
Go to folder:

> `cd client`

Install dependencies:

> `npm install`

Launch Front-end:

> `npm run start`



### Back
Go to folder:

> `cd server`

 Install maven dependencies:

> `mvn clean install`

 Build and run the application: 

> `mvn spring-boot:run`
